# ML-Class-Assignment25
2025机器学习基础课程大作业

## 使用说明 (快速开始)

1.电脑安装 [uv](https://docs.astral.sh/uv/) （一个 python 综合管理工具）

2.在项目根目录运行以下命令：

```
uv run 1_digit/train_pytorch.py
```

uv 会自动安装相关依赖并运行程序（如果你没有python也可以自动安装python），输出模型参数文件（`pytorch_model.pth`）到 `models` 文件夹中，输出识别准确度数据到控制台。

## 报告

GPT 对这个模型和训练效果生成了一个报告，在 `1_digit/reports` 目录下。
